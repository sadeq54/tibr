## How to use

1. Open https://search.google.com/search-console
2. Select property `https://goldpricesarabia.com`
3. Paste URL in top "Inspect any URL" bar
4. Read status:
   - ✅ **URL is on Google** = indexed
   - ⚠ **URL is not on Google** = not indexed → click **Request Indexing**
   - 🔍 **Discovered - currently not indexed** = in sitemap, awaiting crawl
   - 📉 **Crawled - currently not indexed** = quality issue, improve content
5. Mark `[x]` in this file once confirmed indexed

GSC manual indexing quota: ~10-12 per day. Spread over week.

Use the **Pages** report (left sidebar) for a bulk view of all indexed/not-indexed URLs without inspecting one-by-one.

## Daily routine

### Monday-Friday (10 URLs each day)
- 10 priority URLs from Tier 0-1
- Click "Request Indexing" on each
- Wait ~30 sec per URL for confirmation
- Mark `[x] ✅` in this file
- Repeat next priority tier next day

### When publishing new content
1. Add URL to `content/news/articles.ts` → deploy
2. Add new entry in this file under "Editorial Articles"
3. IndexNow push:
   ```powershell
   $body = @{ urls = @("https://goldpricesarabia.com/news/NEW-SLUG", "https://goldpricesarabia.com/en/news/NEW-SLUG") } | ConvertTo-Json
   Invoke-RestMethod -Uri "https://goldpricesarabia.com/api/indexnow" -Method Post -Body $body -ContentType "application/json"
   ```
4. Manually inspect + Request Indexing in GSC

### Weekly check
- GSC → **Pages** report → see indexed vs. not-indexed counts
- Goal first month: 50%+ of Tier 0-2 URLs indexed
- Goal first quarter: 80%+ of all URLs indexed
