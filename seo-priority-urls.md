# SEO — Priority URLs for Google Search Console Indexing

> Google Search Console URL Inspection quota: ~10-12 manual indexing requests per day.
> Submit in priority order below. Spread over multiple days.

Last updated: 2026-05-12
Production domain: `https://goldpricesarabia.com`

---

## How to use

1. Open https://search.google.com/search-console
2. Select property `https://goldpricesarabia.com`
3. Top search bar: "Inspect any URL in 'https://goldpricesarabia.com/'"
4. Paste each URL → press Enter → wait inspection load
5. Click **Request Indexing** button
6. Wait "Indexing requested" ✅ confirmation
7. Move to next URL

If quota hit: "Quota exceeded" message. Retry tomorrow.

## Long-tail (let Google discover via sitemap)

Don't manually request — too many URLs, quota wastes. Sitemap + good internal linking handle these:

- All non-tier-1 countries × karats (e.g. `/morocco/gold-price/21k`, `/india/gold-price/24k`)
- All historical year pages (`/historical-gold-prices/2024`, etc.)
- All crypto coin pages (`/cryptocurrency/bitcoin`, etc.)
- Buy-gold sub-types (`/buy-gold/usa/coins`, etc.)
- Best-gold-price per country variants

Google find via sitemap.xml + internal links. Indexes in 1-4 weeks naturally.

## IndexNow (instant push to Bing + Yandex)

Once `INDEXNOW_KEY` env var set + key file deployed:

```bash
curl -X POST https://goldpricesarabia.com/api/indexnow \
  -H "Content-Type: application/json" \
  -d '{
    "urls": [
      "https://goldpricesarabia.com/",
      "https://goldpricesarabia.com/gold-price/24k",
      "https://goldpricesarabia.com/saudi-arabia/gold-price/21k"
    ]
  }'
```

PowerShell version:
```powershell
$body = @{ urls = @(
  "https://goldpricesarabia.com/",
  "https://goldpricesarabia.com/gold-price/24k"
) } | ConvertTo-Json
Invoke-RestMethod -Uri "https://goldpricesarabia.com/api/indexnow" -Method Post -Body $body -ContentType "application/json"
```
